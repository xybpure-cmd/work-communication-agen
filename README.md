# 工作沟通辅助工具 V1.0（前端原型）

一个**单页网页工具**：输入沟通信息，生成**1 版中文回复草案**。

## V1.0 功能范围（当前准确版本）

### 输入（4 个）
1. 收到的消息（可空）
2. 沟通背景
3. 参考材料
4. 我的意图

### 输出（1 个）
- 一版回复草案

### 交互
- 点击「生成回复」：在页面本地生成回复草案（无后端模型调用）。
- 点击「复制」：复制回复草案到剪贴板。

## V1.0 明确不包含

- 不调用 OpenAI API
- 不做后端模型生成
- 不做记忆系统
- 不接钉钉
- 不自动发送
- 不做多版本生成

## 快速开始

1. 安装依赖

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. 启动服务（仅用于托管页面）

```bash
uvicorn app.main:app --reload
```

3. 打开页面

- http://127.0.0.1:8000

## 页面操作（V1.0）

1. 在页面中填写 4 个输入框（收到的消息可留空）。
2. 点击「生成回复」。
3. 在「一版回复草案」中查看并复制结果。

## 原型说明

- 当前阶段为 V1.0 验证原型，使用本地 mock 逻辑生成回复草案，不依赖真实模型 API。
- 当「收到的消息」为空时，输出将按“主动沟通”场景组织。
- 当「收到的消息」有内容时，输出将按“回应消息”场景组织。
