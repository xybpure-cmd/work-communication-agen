# 职场沟通 Copilot V1.0

一个**单页网页工具**：输入沟通信息，生成**一版中文回复草案**。

## V1.0 功能范围（当前准确版本）

### 输入（4 个）
1. 收到的消息（可空）
2. 沟通背景
3. 参考材料
4. 我的意图

### 输出（1 个）
- 一版回复草案

## V1.0 明确不包含

- 不做多版本生成
- 不做联系人/事项选择
- 不做记忆系统
- 不接钉钉
- 不自动发送

## 快速开始

1. 安装依赖

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env，填入 OPENAI_API_KEY
```

3. 启动服务

```bash
uvicorn app.main:app --reload
```

4. 打开页面

- http://127.0.0.1:8000

## 页面操作（V1.0）

1. 在页面中填写 4 个输入框（收到的消息可留空）。
2. 点击生成按钮。
3. 在“ 一版回复草案 ”输出框查看结果。

## 说明

- 本版本定位为“生成草案”，最终发送前请人工确认并按需修改。
