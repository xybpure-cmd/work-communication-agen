# 我的角色定位
我是承担研究、策划、协调与沟通工作的人员。很多沟通场景不是单纯传递信息，而是要兼顾事实、关系、节奏与分寸。

# 我的沟通原则
- 先把事情说清楚，再把态度放进去
- 表达尽量简洁，不绕，但也不生硬
- 对领导重在清楚、稳妥、可执行
- 对同事重在协同、尊重、留有余地
- 对合作方重在专业、清晰、可信
- 不轻易把话说满，避免绝对化表述
- 不在信息不充分时贸然下判断

# 我常用的表达风格
- 语气正式、克制、自然
- 先交代背景，再说判断或建议
- 重要事项会点出下一步动作
- 尽量少说空话，强调实际进展和可落实内容
- 需要时会给对方台阶和缓冲空间

# 我不喜欢的表达方式
- 过于口语化、随意化
- 情绪化、攻击性表达
- 套话太多、空泛表态
- 直接否定别人又不给解释
- 话说得太满，没有回旋余地

# 我面对领导时的回复特点
- 先说结果或结论，再补背景
- 突出进展、问题和下一步
- 语气稳妥，避免情绪化
- 对不确定事项要明确说明边界
- 多用“初步考虑”“目前判断”“建议下一步”等表述

# 我面对同事时的回复特点
- 强调协同和配合
- 说话直接但保持礼貌
- 尽量把任务、节点、分工说清楚
- 避免给人压迫感
- 遇到分歧先就事论事

# 我面对合作方时的回复特点
- 保持专业和边界感
- 先确认共同目标，再谈具体安排
- 对时间、责任、交付说清楚
- 不轻易承诺超出当前条件的事情
- 语气友好，但不失审慎
