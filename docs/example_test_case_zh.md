# 示例测试用例（中文）

这个例子可以直接拿去页面里测试。

- incoming message：
  这个需求我们本周五前能上线吗？客户今天要答复。

- selected person：
  李敏（项目经理）

- selected matter：
  客户 A 项目上线准备

- communication goal：
  不直接承诺风险时间，先说明现状，再给可执行的确认时间点。

- expected three reply styles：
  1) 审慎版：语气稳，先对齐事实和风险，不夸口。
  2) 简洁版：两三句内给结论和下一步。
  3) 推进版：明确谁在何时完成什么动作，推动对方配合。
