# 运行指南（中文，V1.0 前端原型）

本指南对应当前 **V1.0 单页网页原型**。你只需要完成下面 3 步。

## 1）安装依赖

在项目根目录执行：

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

---

## 2）启动服务

> 服务仅用于提供网页，不参与模型推理。

```bash
uvicorn app.main:app --reload
```

---

## 3）打开页面

浏览器访问：

- `http://127.0.0.1:8000`

---

## 页面中你会看到什么（V1.0）

### 输入区（4 个）
1. 收到的消息（可空）
2. 沟通背景
3. 参考材料
4. 我的意图

### 输出区（1 个）
- 一版回复草案

### 按钮（2 个）
- 生成回复
- 复制

---

## 一次最小可用测试（V1.0）

1. 收到的消息：可不填，或填「客户问：这个需求本周能上线吗？」
2. 沟通背景：填当前项目进展和限制条件。
3. 参考材料：填需求文档、会议结论、排期信息等关键内容。
4. 我的意图：填你希望回复达成的沟通目标。
5. 点击「生成回复」。
6. 在「一版回复草案」查看输出，并点击「复制」。

只要页面能稳定生成回复草案，即为运行成功。
